# Task API

A small REST API built like a real backend service: authentication, validation, CRUD endpoints, and automated tests.

## Features
- Auth (token-based)
- CRUD for tasks
- Input validation + error handling
- OpenAPI/Swagger docs
- Unit + integration tests
- CI-ready structure

## Tech Stack
- Python (FastAPI recommended)
- SQLite
- pytest

## Getting Started
1. Clone the repo
2. Install dependencies
3. Run the API locally
